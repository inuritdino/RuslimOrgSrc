#### Write a message on Facebook

Send us a message using our
[FB account](https://www.facebook.com/RuslimOrg/)

#### Make an issue

Create [an issue](https://github.com/inuritdino/RuslimOrg/issues)
using GitHub interface. This way you could:

- request a new feaure
- make a bug report (including technical, stylistic, and typographical errors)
- suggest a new material/topic/category/question to be included
- make a claim on general inconsistencies of the web-site etc.


#### Make a pull request

Create [a pull request](https://github.com/inuritdino/RuslimOrg/pulls)
using GitHub interface to _suggest modifications to the content_. This
way you can make modifications **yourself** and propose them becoming
an author of Ruslim.Org.
